Au fil des années

Légende

Dans cette page, les jeux joués sont présentés par année.

Pour chaque année, on a entre parenthèse le nombre de jeux terminés (sans compter les extensions1).
Puis un sous-titre avec les jeux terminés (nouveaux en gras et refaits) (sans nommer les extensions1) au cours de celle-ci.
On note en exposant et en abrégé la version jouée (hd: history edition, de: definitive, ee: enhanced edition, cr: classic redux, rem: remaster, rem: remake, res: resurrected, GOTY)

En dessous, un tableau contenat la liste des jeux joués (terminés + essayés) avec :

  • le mois où j'ai fini le jeu
  • la machine sur laquelle je l'ai fini (Elspeth, Shodan, TUF, Xerxès, Big PC, Joystick)
  • le store où j'ai le jeu (Steam, GOG, Battle.net)
  • le numéro de run2 sur la version jouée / nombre de runs total pour ce jeu, toutes versions confondues
  • on comptabilise le nombre de sessions faites pour terminer le jeu, pas le nombre total de sessions
  • le genre (FPS, RPG, RTS, FPS-RPG, FPX, Arcade)
  • si c'est une extension
  • le nom du jeu
    • en gras les jeux finis pour la première fois,
    • en souligné, les jeux pour lesquels j'ai rédigé une critique,
  • l'année de sortie de la version originale
  • l'année de sortie de la version sur laquelle j'ai jouée si différente de la version originale
  • un éventuel commentaire

1: Je considère Heart of the Swarm et Legacy of the Void comme des jeux à part entière et non des extensions de Wing of Liberty.
2: On différencie le nombre de run total et le nombre de run sur une version spécifique : R run actuel / runs totaux.
Ainsi, mon 1er run de Blood (R1/1) est sur la version One Unit Whole Blood et mon second (R1/2) est sur la version Fresh supply.
