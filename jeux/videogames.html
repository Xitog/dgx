J'ai surtout joué à trois grands types de jeux vidéo :

  • les FPS,
  • les RTS,
  • les RPG.

Pour chacun de ces types, on peut dater ma rencontre avec un jeu initiateur, parfois double. Pour les FPS, ce fut Duke Nukem 3D. Pour les RTS, ce fut Starcraft. Pour les RPG, ce fut Diablo pour les action-RPG et Baldur's Gate.
